A short note on the materials of this site. It is set in two typefaces, lives on four colors, and was built with care — the way I want to build most things.

Typefaces
Mona Sans for everything that carries meaning. JetBrains Mono for dates, tags, and small metadata.
Palette
Paper Ink Muted Mustard
Mustard is reserved for the reading-progress bar. Nothing else.
Mark
A kanji seal, designed as a small stamp. The mark in the margin is two nested squares, deliberately offset.
Grid
680px reading column, 72px left gutter. Dates and anchors live in the gutter. The body column stays quiet.
Built with
Hand-written HTML, CSS, and a little JavaScript. No framework. No build step. Hosted on GitHub Pages.
Philosophy
Quiet, spacious, and built deeply. For the AI era, those feel like the things worth keeping.
Thanks
To everyone whose writing made me want to start one of these.
